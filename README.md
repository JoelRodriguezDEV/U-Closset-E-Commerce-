## üíé About The Project

**U CLOSSET** is a modern e-commerce frontend designed around the concept of **"Silent Luxury"**. The interface prioritizes minimalism, typography, and negative space over clutter, delivering a high-end shopping experience similar to top fashion houses.

Built with **React (Vite)** and **TypeScript**, this project focuses on performance, accessibility, and fluid micro-interactions.

### ‚ú® Key Features

- **Silent Luxury Aesthetic:** A monochromatic, editorial design that puts the product in focus.
- **Advanced UX/UI Patterns:**
  - **Expandable "Pill" Search:** A non-intrusive search bar that expands dynamically without shifting layout.
  - **Glassmorphism UI:** Floating cart and elements with frosted glass effects.
  - **Smart Navbar:** Adapts transparency based on scroll position and route.
- **Full User Dashboard:** A gamified "My Account" section inspired by top tiers e-commerce platforms (Orders tracking, Wallet, Points).
- **Dark/Light Mode:** Fully integrated theme switching with persistence.
- **Responsive Design:** Optimized for all devices, with a custom mobile drawer navigation.

## üöÄ Getting Started

1.  **Clone the repo**
    ```bash
    git clone [https://github.com/tu-usuario/u-closset-frontend.git](https://github.com/tu-usuario/u-closset-frontend.git)
    ```
2.  **Install packages**
    ```bash
    npm install
    ```
3.  **Setup Environment Variables**
    Create a `.env` file in the root directory:
    ```env
    VITE_API_URL=http://localhost:3000
    ```
4.  **Run Development Server**
    ```bash
    npm run dev
    ```
